# n8n Lead Automation Suite

A modular lead management and qualification automation system built using n8n.

## Overview

This repository contains production-ready n8n workflows designed to:

- Capture incoming leads via webhook
- Enrich and classify leads
- Prioritize high-value business emails
- Trigger automated notifications
- Log structured lead data into Google Sheets
- Send intelligent follow-ups

The system is designed as a scalable LeadOps foundation.

## Workflow Architecture

1. Lead Capture & Qualification Automation  
2. AI Lead Qualification System  
3. Lead Enrichment & Notification System  
4. Priority Lead Notification Automation  
5. Lead Auto Response & Follow-up Automation  
6. Lead Logging Workflow  

Each workflow is modular and can operate independently or as part of a larger automation pipeline.

## Tech Stack

- n8n
- Webhooks
- JavaScript (custom nodes)
- Gmail API
- Google Sheets API

## Use Cases

- Startup lead routing
- Sales automation
- Founder notification systems
- AI-assisted lead scoring
- CRM-lite pipeline systems
